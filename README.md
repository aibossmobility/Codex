# Papa Life Website

This working copy includes the Papa Life AI Coach website integration.

## Papa Life AI

Main implementation files:

- `client/src/components/PapaAiWidget.tsx`
- `client/src/pages/PapaAiExperience.tsx`
- `server/papa-ai-engine.ts`
- `server/papa-life-system-prompt.md`
- `docs/papa-life-ai-experience.md`

Primary API routes:

- `GET /api/health`
- `GET /api/ai/status`
- `POST /api/ai/chat`
- `POST /api/ai/assessment`
- `POST /api/ai/leads`
- `POST /api/leads`

See `docs/papa-life-ai-experience.md` for environment variables, deployment steps, and the testing checklist.
